Biopython used to have static copies of various example files used in the Biopython Tutorial (PDF) online under this folder. These were manually kept in sync with the main Biopython source code.

This part of the website is now part of the Biopython DIST GitHub Repository at github.com/biopython/DIST using GitHub Pages.

Please refer directly to https://github.com/biopython/biopython/tree/master/Doc/examples and update any URLs appropriately (using the raw links as needed).

Thank you.
